有三个页面：

  • a.com/app.html：应用页面。
  • a.com/proxy.html：代理文件，一般是一个没有任何内容的html文件，需要和应用页面在同一域下。

  • b.com/data.html：应用页面需要获取数据的页面，可称为数据页面。
